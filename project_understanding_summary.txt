3. 技術スタック

------------------------------------------------------------------

- Vite 6 マルチページビルド

- HTML + CSS + バニラ JS（ES module）

- CMS: JSON import → script.js が DOM 描画

- 多言語: cms/i18n.json + CMS内 ja/en/zh + localStorage（全テンプレ共通）



------------------------------------------------------------------

4. npm スクリプト & URL

------------------------------------------------------------------

- npm run dev → /templates/（テンプレ一覧・2列グリッド）, /template/,

               /template/pet-friendly-ryokan/, /template/traditional-kominka-inn/,

               /template/luxury-glamping/, /template/ocean-view-onsen-ryokan/

- npm run build → dist/ に7ページ以上出力

- メインLP（index.html）: 920px / 768px / 560px / 400px ブレークポイントでレスポンシブ対応

  （ヒーロー・各セクション・ドロワーナビ・フォーム・フッター）

- メインLPフッター: 株式会社コイシカワ情報（住所・電話・コーポレートリンク）、

  会社概要欄はURL（https://koishikawa.co.jp/）のみ、

  法的情報リンク（プライバシーポリシー・個人情報の取扱・労働方針声明）

- モバイルヘッダー（920px以下）: 料金・FAQ・お問い合わせリンクを非表示

  （ハンバーガーメニュー内からも除外。無料相談CTAは残す）

- ヘッダーブランド: KOISHIKAWA ロゴのみ（サブ文言なし）

- ヒーロー見出し: 「旅館・ホテル業界 専門 Webサイト制作」（バッジ風・小さめ表示）

- ヘッダーナビ: ロゴ右のナビは flex で横幅いっぱいに均等配置、CTA ボタンは右端。

  ナビ・ボタンともやや大きめ（14px / padding 拡大）で間隔を均等化（2026-06-23）

- お問い合わせ・テンプレ購入通知先: koishikawa.vibecoding@gmail.com

  - LP フッター・お問い合わせセクションにメール記載

  - テンプレ購入ボタン → #contact フォーム（テンプレ名・プラン自動入力）

  - 問い合わせ API: `/api/send-inquiry`（Vercel Serverless）

  - Supabase inquiries テーブルに保存 + Resend で Gmail 通知

  - Vercel 必須 env: SUPABASE_SERVICE_ROLE_KEY（supabase/README.md）

6. 写真表示（2026-06-23 更新）

------------------------------------------------------------------

- 5テンプレートの CMS セクション（客室・プラン・料理・アクティビティ・周辺観光）は、

  JavaScript 読み込み前でも写真が表示されるよう HTML に静的フォールバックを追加。

  JS 起動後は従来どおり CMS JSON で上書き（多言語切替も維持）。

- テンプレート一覧（templates/index.html）のミニプレビューは、

  グラデーションのみだったヒーロー・客室カード・温泉/庭園エリアに Unsplash 背景写真を追加。

- 写真ソース: Unsplash CDN（cms/*.json と同一 URL）。ローカル画像ファイルは未コミット。

- 表示確認: npm run dev → http://localhost:5173/templates/ および各 /template/ 配下
