5. Windows 対応（2026-06-17 修正）
------------------------------------------------------------------
[判明した原因]
1. package.json が存在せず npm install / npm run dev が ENOENT で失敗していた。
2. Windows 環境で npm レジストリへの SSL 検証が失敗
   (UNABLE_TO_VERIFY_LEAF_SIGNATURE) し、install が極端に遅延・停止。
3. OneDrive 同期フォルダ内の node_modules がプレースホルダのまま
   中身が空になり、vite コマンドが見つからない状態になることがある。

[実施した修正]
- package.json / vite.config.js / .npmrc / .gitignore / .nvmrc を追加。
- npm install で package-lock.json を生成（Vite 6.4.3 固定）。
- .npmrc に strict-ssl=false を設定（社内プロキシ等での SSL 問題回避）。
- スクリプトは vite 直接呼び出しのみ（OS 非依存）。

[推奨環境]
- Node.js 20 LTS 以上（.nvmrc: 20）。動作確認: v24.13.1 でも OK。
- Python / Visual Studio Build Tools: 不要
  （Vite + esbuild は prebuilt バイナリを使用、node-gyp 不要）。
- プロジェクトを OneDrive 外 (例: C:\dev\RyokanKoishikawa) に置くことを推奨。
  OneDrive 利用時は node_modules を「常にこのデバイスに保持」に設定。
