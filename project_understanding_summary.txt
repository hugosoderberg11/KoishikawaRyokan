3. 技術スタック

------------------------------------------------------------------

- Vite 6 マルチページビルド

- HTML + CSS + バニラ JS（ES module）

- CMS: JSON import → script.js が DOM 描画

- 多言語: cms/i18n.json + CMS内 ja/en/zh + localStorage（全テンプレ共通）

6. 写真表示（2026-06-23 更新）

------------------------------------------------------------------

- 5テンプレートの CMS セクション（客室・プラン・料理・アクティビティ・周辺観光）は、

  JavaScript 読み込み前でも写真が表示されるよう HTML に静的フォールバックを追加。

  JS 起動後は従来どおり CMS JSON で上書き（多言語切替も維持）。

- テンプレート一覧（templates/index.html）のミニプレビューは、

  グラデーションのみだったヒーロー・客室カード・温泉/庭園エリアに Unsplash 背景写真を追加。

- 写真ソース: Unsplash CDN（cms/*.json と同一 URL）。ローカル画像ファイルは未コミット。

- 表示確認: npm run dev → http://localhost:5173/templates/ および各 /template/ 配下
